About the course

This course is designed for those who wish to become quality assurance engineers.No prior knowledge of programming is required. During the course, you'll learn basic concepts of testing. As a quality assurance engineer you will explore the principles of working with Agile and Waterfall, will have a complete insight into the stages and types of testing, and will apply that knowledge to software testing. In the final part of the course, you will explore web-based automated testing using Javascript and Selenium. Course program has been based on requirements of leading IT companies, which will guarantee your future employment.

Education


Lectures and seminars are important components of our teaching method. After each lesson, students receive homework and lecture materials, which are available online in their personal account. Course Syllabus

IT team members, roles and responsibilities
Fundamentals of Testing, Test process and main activities
Software development models(Waterfall, V-model, etc.), the QA role in SDLC
Test types, Test levels
Introduction to AGILE, SCRUM methodology
Test design techniques, categories
Black-box/White-box testing, types and techniques
Test Management(planning and monitoring)
Introduction to Client-server architecture, API testing and GIT version control system
Introduction to Web Browser Automation with Selenium WebDriver
Practice of writing Test cases and Bug reporting using appropriate Testing tools(Jira, TestRail, etc.)
Testing experience on real projects
